This API can be used by the Alliance Partner/Aggregator to inquire a merchant under it using the unique Merchant ID assigned to it. Response will give all the details of that merchant.

API URL api/allp/application/submit
HTTP Method POST
REQ/RES TYPE Application/JSON
Field name Description Data type Mandatory Length
Message ID Message ID for application submit request String
Object Object content –
Application status Specifies the application status “SUBMITTED” Enum
ID Specifies the request ID String Y
Request type CREATE (Fixed value) String
